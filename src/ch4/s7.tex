\wde{4.7 Cost per Time Unit} Let $c(i)$ be the cost per time unit spent at state $i$. Then the long run average cost per time unit is 
$\psi_i = \lim_{T \to \infty} (^1/_T) E(\int^T_0 c(X_t)\ dt | X_0 = i)$.
\wt{4.7.1} For an irreducible positive recurrent chain $\psi_i = \sum_{j \in S} \pi_j c(j).$
