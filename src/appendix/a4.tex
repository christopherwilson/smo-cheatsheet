\wde{Uniform 4.4} Let $X$ be uniform on $[a,b]$, $f_X(x) = 1/(b-a)$ for $a < x < b$, and 0 otherwise. $\E(X) = (b+a)/2$, $\Var(X) = (b-a)^2/12$.
\wde{Normal 4.4} Let $X \sim \mathrm{normal}(\mu, \sigma^2)$, then $f_X(x) = \frac{1}{\sqrt{2\pi\sigma^2}}e^{-(x-\mu)^2/(2\sigma^2)}$, $\E(X) = \mu$, $\Var(X) = \sigma^2$.
\wde{Standard Normal 4.4} Standard Normal is $\mathrm{normal}(0,1)$, $f(x) = \frac{1}{\sqrt{2\pi}}e^{-x^2/2}$, $\E(X) = 0$, $\Var(X) = 1$. $\mu + \sigma X$ is $\mathrm{normal}(\mu, \sigma^2)$.
\wde{Exponential 4.4} Let $X \sim \mathrm{exponential}(\lambda),$ then $f(x) = \lambda e^{-\lambda x}$ for $x > 0$, and $0$ otherwise. $\E(X) = 1/\lambda$, $\Var(X) = 1/\lambda^2$.
\wde{Gamma 4.4} Let $X \sim \mathrm{gamma}(n,\lambda) = \sum^n_{i=1} \mathrm{exponential}(\lambda)$, then $f(x) = \lambda e^{-\lambda x}(\lambda x)^{n-1}/(n-1)!$ for $x > 0$, and 0 otherwise, $\E(X) =n/\lambda$, $\Var(X) = n/\lambda^2$.
\wt{Distributing $\E$, $\Var$} For both discrete and continuous RVs: $\E(b + aX) = b+a\E(X)$ and $\Var(b+aX) = a^2\Var(X)$
